# LightEnvReconstruction
概要

360度画像から光源の方向、強度、色を推定し、推定された各光源情報をUnityのDirectional Lightに割り当て、画像の光源環境を再構築する。

結果

背景とパネルに乗せた３６０度画像から光源環境を再構築した。
